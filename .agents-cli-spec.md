# Agent Spec

## Overview
The UTCI Tracker Agent is a local prototype built using the Google ADK (Agent Development Kit). Its purpose is to respond to user queries about spatial and temporal thermal comfort and heat stress trends in Kerala, India. It queries a PostgreSQL database table (`utci_grid`) that holds Universal Thermal Climate Index (UTCI) values mapped to spatial coordinates, districts, and taluks in Kerala across an 8-day window (the last 7 days plus the current day's observations at 1:30 PM IST / 08:00 UTC).

## Example Use Cases
*   **Spatial Query:** "Which taluk in Thrissur district is experiencing the most intense thermal discomfort today?"
    *   **Agent Action:** Queries `utci_grid` for the maximum `utci_value` grouped by `taluk` in the district of "Thrissur" for the current date.
*   **Temporal Trend Query:** "How has the average UTCI value for Ernakulam district changed over the last week?"
    *   **Agent Action:** Queries `utci_grid` to get `AVG(utci_value)` grouped by `observation_date` where `district = 'Ernakulam'`, sorted chronologically.
*   **Threshold Exceedance Trend:** "How many km2 of area in Kerala has experienced UTCI values above 35 degrees celsius daily over the last 7 days?"
    *   **Agent Action:** Queries `utci_grid` to sum `pixel_area_km2` where `utci_value > 35.0`, grouped by `observation_date`.

## Tools Required
*   **Postgres Database Query Tool:** A tool that allows the agent to execute read-only SQL queries on the `kerala_utci` PostgreSQL database (specifically the `utci_grid` table) to retrieve thermal comfort stats.

## Constraints & Safety Rules
*   **Read-Only Database Access:** The agent must only run read-only queries (`SELECT`). It must NOT execute queries that modify data (`INSERT`, `UPDATE`, `DELETE`, `DROP`, `ALTER`).
*   **Local Prototype:** The agent runs locally on the user's machine (no Cloud deployment or external GCP APIs are used for database hosting).
*   **Kerala Boundary:** Queries must be restricted to data within Kerala's boundaries.

## Success Criteria
*   The agent successfully translates natural language queries about thermal comfort/UTCI spatial distribution and temporal trends in Kerala into SQL queries.
*   The agent executes these SQL queries on the local PostgreSQL database using its database tool.
*   The agent provides accurate answers and trends to the user based on the query results.
